稍微整理下自己的常用命令

教程
实例

常用

ctrl + L 清除屏幕
ctrl + U 前删
ctrl + K 后删
ctrl + E 行尾
ctrl + A 行首
shift + pageup 上翻
shift + pagedown 下翻
env 环境变量路径
chmod 权限
su 用户切换
sudo root权限执行
apt-get 安装应用
tar -xvf 解压tar格式 unzip加压zip格式

目录

/bin/sh bash 等脚本执行 命令
/boot 核心启动文件
/dev 设备
/etc 配置文件，目录
/lib 动态链接库
/usr 用户程序目录
pwd 当前目录
ls 查看目录内容 -a 查看隐藏文件
/ root根目录 ~ 用户目录

端口 进程

top 实时进程
lsof 端口
netstat -tunlp
screen 多进程复用一个窗口
screen -ls 查看所有的窗口 可以恢复
ifconfig 查看ip

编辑文本

awk 格式化文本常用
sed 编辑到匹配的文本
grep 查找文本
egrep 扩展正则
vim 编辑文本 i进入编辑模式 esc退出 :wq 保存退出 q!强制退出
