读书

如果专业书籍也算的话，那就是 数据密集型应用系统设计 与 嵌入式c语言自我修养 一本是大量数据的数据库以及分布式设计相关的 一本是偏向于一些硬件底层的linux，c相关的。 只是起了个头，主要是没有太多时间去看，最近密集找工作快两个星期了，疯狂看一些面试八股跟练习一些算法相关的内容。

理财

新开的一个栏目。我大概是从疫情前的一段时间开始炒股，理财的。近几年来也算是有一点收益，但不是很多。在股票方面还算是一个刚入门的菜鸟，所以个人投资方面还是偏向于稳定的。大部分用来r2中低风险的稳定理财，一年有3-4%的收益，小部分用来炒股。

最近在研究可转债以及提前埋伏可转债，或许有一点汤可以喝。最近大盘不断走高，总感觉风险逐步在接近，打算减仓一部分。最近还在看一些etf，集合竞价相关的知识。

影视

最近没看电影，看了电视剧 扫黑风暴，可以说是高开低走的一部剧，后面的剧情特别赶进度，有传言是有大量删减，真实情况不得而知，有可能只是单纯的剧组控制不住后面的节奏，毕竟之前看过的很多影视作品或者动画，高开低走的现象太过普遍了。

杂谈

最近面试了挺多次，可能快有十几次了，随便谈谈感想。首先是规模越大的企业流程上来说更加的正规，面试官的整个水准也会更高，会仔细去看简历里提到的东西，甚至是开源项目的代码。面试的时候会交流对于这些项目的看法。当然面试一些小的公司时候，也会有非常细致认真的面试官。单单是这样的几次简短的交流，就已经发现了之前做的开源项目在很多方面的漏洞，以及对于不同情况下的处理存在很多问题。也算是受益匪浅了。更关键的是每面一次，也算是发现自己不足的一些地方，这些都是之前蒙头写代码很难发现的。

游戏

最近电脑上的游戏没有在玩，手游还是刷刷每日，平均每天半小时不到。

中间有过一段时间在玩 弹射世界 但后来弃坑了，感觉没啥意思，刷每日或者活动都挺花时间的，关键是游戏角色池子太深了，很多角色要去抽，顶不住。
